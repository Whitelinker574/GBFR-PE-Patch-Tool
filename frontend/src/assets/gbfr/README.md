# GBFR UI illustration assets

Official in-game and character-page references retained by the project were used for identity, outfit, and palette checks.

`journal-scene-4k.webp` is the washed 16:9 notebook banner used on the home screen. Function pages use the character-free ornamental `parchment-ui-v2.webp`; transparent character art under `cutouts/` is composited as a fixed background layer. Only assets referenced by the production interface are kept in this directory.

The illustrations are fan-made UI assets, not official game artwork. Granblue Fantasy and its characters are properties of Cygames, Inc. Confirm redistribution and trademark rights before redistributing these assets.
